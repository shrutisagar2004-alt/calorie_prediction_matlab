🍽️ AI Food Detection & Calorie Estimation System

An AI-powered MATLAB application that detects food items from images using the pre-trained ResNet-50 deep learning model and estimates calorie intake based on portion size. The system also calculates Body Mass Index (BMI) and Basal Metabolic Rate (BMR) to provide personalized nutritional insights and healthy food recommendations.

📌 Features
Detects food items using the pre-trained ResNet-50 model.
Calculates BMI and BMR from user information.
Estimates daily target calorie intake based on fitness goals.
Predicts food quantity using image processing techniques.
Calculates calories, carbohydrates, protein, and fat.
Suggests healthy alternative food options.
Compares consumed calories with the target calorie requirement.
Displays graphical calorie comparison for dietary analysis.
